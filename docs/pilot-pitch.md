# Winefeed

**Från offert till leverans – med full kontroll på compliance och ansvar**

---

## Problemet

### Så ser det ut idag

En restaurang vill ha vin från en europeisk producent. Vad händer?

| Steg | Verklighet |
|------|------------|
| Förfrågan | Mail fram och tillbaka, oklara specs |
| Offert | Excel-fil, manuell prissättning, svårt att jämföra |
| Order | Kopierat från offerten, ofta fel |
| Import | Vem ansvarar? Vilka dokument saknas? När kommer DDL-beslutet? |
| Leverans | "Var är mitt vin?" – ingen vet |

**För importören innebär detta att varje affär kräver manuellt arbete, ständig uppföljning och hög risk för fel.**

**Resultatet:**
- Veckor av väntan
- Fel i dokumentation → förseningar i tullen
- Irriterade restauranger
- Importörer som drunknar i manuellt arbete

---

## Lösningen

### Winefeed: En plattform, hela flödet

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   REQUEST   │ ──▶ │    OFFER    │ ──▶ │    ORDER    │ ──▶ │   IMPORT    │
│             │     │             │     │             │     │    CASE     │
│ Restaurang  │     │ Leverantör  │     │ Bekräftad   │     │ IOR hanterar│
│ beskriver   │     │ svarar med  │     │ av båda     │     │ compliance  │
│ sitt behov  │     │ konkret     │     │ parter      │     │ automatiskt │
│             │     │ offert      │     │             │     │             │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
```

**Alla parter ser samma status i realtid – restaurang, leverantör och importör. Inget faller mellan stolarna.**

## Före och efter

| Moment | FÖRE (mail + Excel) | EFTER (Winefeed) |
|--------|---------------------|------------------|
| **Skicka förfrågan** | Mail till 5 leverantörer, vänta på svar | 1 klick → rätt leverantörer får den automatiskt |
| **Få offert** | Excel-bilagor, olika format, svårt att jämföra | Strukturerade offerter, jämförbar data |
| **Bekräfta order** | Kopiera-klistra, riskerar fel | Offert → Order med ett klick, inga fel |
| **Compliance-status** | "Har vi DDL? Vilka dokument saknas?" | Tydlig status: ✅ OK / ⚠️ Saknas / ❌ Blockerad |
| **Spårbarhet** | Mail-trådar, vem sa vad? | Komplett historik, alla händelser loggade |
| **Leveransuppföljning** | "Ring lagret och fråga" | Real-time status i systemet |

## Vad detta betyder för dig som importör

### Mindre manuellt arbete
- Sluta kopiera data mellan system
- Sluta jaga dokument via mail
- Sluta svara på "var är min leverans?"

### Färre fel
- Strukturerad data från start
- Validering innan order bekräftas
- Compliance-checkar innan det är för sent

### Nöjdare restauranger
- De ser att något händer
- De vet vem som ansvarar
- De får sina leveranser i tid

### Trygghet i compliance
- DDL-status synlig för alla parter
- Dokument samlade på ett ställe
- Spårbarhet om Tullverket frågar

## Vad Winefeed inte är

- **Inte ett nytt affärssystem** – ni behåller era befintliga verktyg
- **Inte ett lager- eller ERP-verktyg** – vi hanterar flödet, inte lagersaldot
- **Inte en marknadsplats** – vi tar inte över kundrelationen

Winefeed är ett operativt flöde ovanpå era befintliga processer.

---

## Varför pilot nu?

Regelverk skärps, fler små producenter vill in på marknaden, och kraven på spårbarhet ökar. Manuella arbetssätt skalar inte längre.

Winefeed är byggt för just den här verkligheten.
